base_model

In this module we have the base model, learner and other things we need to train encoder

Here we have the base functions and classes to train a basic BT-style model. Note that this (mostly) all comes directly from here: https://github.com/KeremTurgutlu/self_supervised/blob/main/nbs/14%20-%20barlow_twins.ipynb but we needed to extend some of the functionality for our purposes.

Some of the base classes and functions needed for image augmentation pipeline:

Here we write down standard definition of lf for RAT method:

BarlowTwins needs an lf method to work properly. Here we provide the lf of standard barlow twins. Later we can patch in a new defintion of lf that involves random functions, inner maximization etc. The tools needed to do this are provised in base_lf
